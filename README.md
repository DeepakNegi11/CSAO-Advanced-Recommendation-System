# CSAO-Advanced-Recommendation-System
Zomato Cart Super Add‑On (CSAO) recommendation system built for the Zomato hackathon using FastAPI, LightGBM, and Streamlit. Ranks top‑5 add‑on items per cart to boost Average Order Value and Cart‑to‑Order, with cold‑start handling, contextual features, and an interactive dashboard.
